## tsuru role add

Create a new role for the specified context type.

### Synopsis

Create a new role for the specified context type.
Valid context types are:

* global
* app
* team
* user
* pool
* service
* service-instance
* volume
* router
* job

The [[--description]] parameter sets a description for your role.
It is an optional parameter, and if its not set the role will only not have a
description associated.


```
tsuru role add <role-name> <context-type> [--description/-d description] [flags]
```
